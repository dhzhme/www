---
title: 个人常用快捷键收集
tags: [Visual Studio Code]
---

| Keyboard shortcut | Remark                                 |
| :---------------- | :------------------------------------- |
| ⌘ + ⏎             | add a line down                        |
| ⇧ + ⌘ + ⏎         | add a line up                          |
| ⇧ + ⌘ + P         | show all commands                      |
| ⌥ + ⌘ + L         | find in selection                      |
| ⇧ + ⌥ + ↑         | copy line up                           |
| ⇧ + ⌥ + ↓         | copy line down                         |
| ⌘ + D             | multiple choice                        |
| ⇧ + ⌘ + [         | previous tab                           |
| ⇧ + ⌘ + ]         | next tab                               |
| ⌥ + ⌘ + Z         | format document in VSCode/JetBrains    |
| ⌥ + ⌘ + K         | Bookmarks Extension: add a bookmark    |
| ⌥ + ⌘ + J         | Bookmarks Extension: previous bookmark |
| ⌥ + ⌘ + L         | Bookmarks Extension: next bookmark     |
